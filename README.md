Transcription App

This is a simple React app that uses Web Speech API to transcribe spoken words into text.

# Features

- Start and stop recording with microphone icon
- Transcript display once the recording is done
- Able to copy the transcripted text to clipboard with copy button
- Built with React, TypeScript and Vite

# Usage

- Click on the microphone below the title
- Speak into your mic (in english)
- Click on the microphone again to stop the recording and watch the transcripted text appear
- Click on the copy button to copy text to clipboard
